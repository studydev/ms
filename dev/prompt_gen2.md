# PURPOSE
Create a high-resolution, ICON-FORWARD enterprise INFOGRAPHIC for an executive
slide deck. The infographic explains how four intelligence layers converge
into one unified operating system called "Microsoft IQ". Audience: enterprise
IT decision makers. Visual medium: clean flat vector illustration with subtle
gradients and soft glow, corporate Microsoft-like design language
(Segoe-style sans-serif), minimal but polished. Photorealism is NOT allowed.
The design philosophy is "ICON-FIRST": recognizable Microsoft, Azure, and
GitHub product icons dominate the visual; text is kept short so icons read at
a glance.

# BACKGROUND (scene & canvas)
- Aspect ratio: 3:2 (landscape, 1536x1024).
- Background: pure white (#FFFFFF) with a very faint radial glow behind the
  center. No textures, no noise, no decorative patterns.
- Thin neutral cross-divider separates the canvas into four equal quadrants
  (top-left, top-right, bottom-left, bottom-right).
- A luminous circular CORE sits at the exact intersection of the four
  quadrants, overlapping all four. Four colored energy ribbons flow from each
  quadrant INTO the core, expressing "4 layers → 1 OS".

# SUBJECT (what must appear)
Four quadrants + one central core. Each quadrant is DOMINATED by a compact
constellation of 6–8 LABELED PRODUCT-ICON BADGES arranged around a faint
central metaphor illustration. Each badge = a rounded-square tile holding the
product's recognizable glyph (in its own brand color) with the product name
label beneath. Icons are the hero; text copy is secondary.

# DETAILS — COLOR TOKENS (quadrant accents — MICROSOFT LOGO PALETTE)
The four quadrant accent colors map 1:1 to the Microsoft corporate logo's
four squares, preserving the exact corner positions (red top-left,
green top-right, blue bottom-left, yellow bottom-right). This reinforces
that Microsoft IQ is the unifying operating system behind all four layers.

- TOP-LEFT   (Work IQ):    Microsoft RED     #F14F21  (+ deep coral #C43A14)
- TOP-RIGHT  (Fabric IQ):  Microsoft GREEN   #7EB900  (+ deep olive #5C8A00)
- BOTTOM-LEFT(Foundry IQ): Microsoft BLUE    #00A3EE  (+ deep azure #007ACC)
- BOTTOM-RIGHT(SDLC IQ):   Microsoft YELLOW  #FEB800  (+ deep amber #C78E00)
- Central core: blended glow of all four Microsoft logo colors, soft white
  halo, faint circuit/neural pattern inside.
Each product icon keeps its OWN signature brand color on its badge
(e.g., Outlook blue, Excel green, Teams purple, PowerPoint orange, Power BI
yellow, GitHub graphite, Azure blue). Badge borders, title color, tagline
color, and the quadrant's soft tint wash all derive from that quadrant's
Microsoft-logo accent color.
